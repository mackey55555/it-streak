# キャラクター画像生成プロンプト

## 基本キャラクター設定（全プロンプトに共通）
- チビキャラ（デフォルメスタイル）
- 狐のような特徴（フードから出た耳、ふわふわの尻尾）
- 淡い緑色の髪（ミディアムレングス、前髪あり）
- クリーム色（薄いベージュ）のフード付きパーカー
- パーカーの腹部にオレンジ色の炎のようなシンボルマーク
- 淡い緑色のズボン
- 大きく輝く茶色の瞳
- アニメ風のイラスト
- 白背景または透明背景
- 優しく温かみのある色合い

---

## Phase 1: 基本シーン

### 1. 正解時のフィードバック（クイズ画面）
**ポーズ**: 正面を向いて立っている
**表情**: 大きく笑顔、目を細めて喜んでいる
**アクション**: 両手を上げて拍手している、または両手を上げて喜んでいる
**サイズ感**: 中サイズ（100px x 120px程度）
**背景**: 透明

**プロンプト**:
```
チビキャラ（デフォルメ）、正面を向いて立っている、大きく笑顔で目を細めて喜んでいる表情、両手を上げて拍手している、狐のような耳とふわふわの尻尾、淡い緑色の髪、クリーム色のフード付きパーカー、パーカーの腹部にオレンジ色の炎のシンボルマーク、淡い緑色のズボン、大きく輝く茶色の瞳、アニメ風のイラスト、透明背景、優しく温かみのある色合い、正解を祝福している様子
```

### 2. 不正解時のフィードバック（クイズ画面）
**ポーズ**: 正面を向いて立っている

**アクション**: 片手を胸の前、または両手を前に出して優しく励ましている
**サイズ感**: 中サイズ（100px x 120px程度）
**背景**: 透明

**プロンプト**:
```
チビキャラ（デフォルメ）、正面を向いて立っている、優しい励ましの表情（少し悲しそうだが前向き、優しく微笑んでいる）、片手を胸の前または両手を前に出して優しく励ましている、狐のような耳とふわふわの尻尾、淡い緑色の髪、クリーム色のフード付きパーカー、パーカーの腹部にオレンジ色の炎のシンボルマーク、淡い緑色のズボン、大きく輝く茶色の瞳、アニメ風のイラスト、透明背景、優しく温かみのある色合い、励ましと応援の様子
```

## ファイル命名規則

生成した画像は以下の命名規則で保存してください：

1. character-correct.png（正解時）
2. character-incorrect.png（不正解時）
3. character-result-high.png（結果80%以上）
4. character-streak-side.png（ストリークカード右上）
5. character-progress-complete.png（進捗目標達成）
6. character-result-good.png（結果60-79%）
7. character-streak-celebration.png（ストリーク継続）
8. character-explain.png（解説表示時）
9. character-progress-ongoing.png（進捗未達成）
10. character-result-medium.png（結果40-59%）
11. character-result-low.png（結果40%未満）
12. character-loading.png（読み込み中）
13. character-welcome.png（初回ユーザー）
14. character-empty.png（問題なし）
15. character-error.png（エラー時）
